# Installation Guide

Welcome to the Git Homework Repository! This guide will help you set up the project on your local machine.

## Prerequisites

Before you begin, make sure you have the following installed:

- **Git**: Version control system ([Download Git](https://git-scm.com/downloads))
- **Web Browser**: Chrome, Firefox, Edge, Safari, or any modern browser

## Installation Steps

Follow these steps to clone the repository and view the webpage:

### 1. Clone the Repository

Open your terminal or command prompt and run the following command to clone the repository:

```bash
git clone https://github.com/SE-Fall24-G73/git-homework.git
```


### 2. Navigate to the Project Directory

Change your current directory to the project folder:

```bash
cd git-homework
```

### 3. Open the Webpage

You can view the team webpage by opening the `index.html` file in your web browser.

#### Option A: Using the Command Line

- **For macOS/Linux:**

  ```bash
  open index.html
  ```

- **For Windows:**

  ```bash
  start index.html
  ```

#### Option B: Manually Open the File

- Navigate to the `git-homework` directory using your file explorer.
- Double-click on `index.html` to open it in your default web browser.

## Optional Steps

### Editing the Webpage

If you want to make changes to the webpage:

1. Open `index.html` in a text editor (e.g., Visual Studio Code, Sublime Text, Notepad++).
2. Make your desired changes.
3. Save the file and refresh the webpage in your browser to see the updates.

### Running a Local Server (Optional)

For a better development experience, you can run a local web server:

- **Using Python 3:**

  ```bash
  python -m http.server
  ```

  Then open `http://localhost:8000` in your web browser.

## Troubleshooting

- **Git Command Not Found**

  If you receive an error saying `git` is not recognized, ensure that Git is installed and added to your system's PATH.

- **Webpage Not Displaying Properly**

  - Verify that all files have been cloned correctly.
  - Check for any errors in the console of your web browser's developer tools.
